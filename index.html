Dungeons and Dragons Character Generator

I created a dynamic website using a full MEAN stack using Mongoose to give my database schemas. The purpose of this app is to make simple the process of creating a new DnD character for someone who has never played before, and give them enough knowledge that they will be able to play.

Attendance Tracker

As part of a group of four using SCRUM and pair programming, we created a website for tracking attendance in a class. Built with ruby on rails, it includes API calls to send and recieve texts and emails so that students can let the app know they will be late in advance, and get notifications when they are accruing too many absences.

Delivery Database
